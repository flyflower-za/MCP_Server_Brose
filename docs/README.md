## 📝 文档贡献

### **如何贡献文档**

1. **创建新文档**
```bash
# 在 docs/ 目录下创建
vim docs/NEW_FEATURE.md

# 更新索引（本文件）
# 添加链接和说明
```

2. **更新现有文档**
```bash
# 编辑文档
vim docs/EXISTING_DOC.md

# 更新 CHANGELOG.md
# 记录变更
```

3. **文档规范**
- 使用 Markdown 格式
- 添加清晰的标题和目录
- 提供代码示例
- 包含故障排除部分
